# Documentación Legal - Playas RD

Esta carpeta contiene los documentos legales de la aplicación Playas RD, publicados en GitHub Pages.

## Archivos

- `index.html` - Página principal con enlaces a los documentos
- `politica-privacidad.html` - Política de Privacidad
- `terminos-servicio.html` - Términos y Condiciones de Uso

## Configurar GitHub Pages

Para publicar estos documentos en GitHub Pages:

1. Ve a tu repositorio en GitHub: `https://github.com/yohan201609-oss/playa-rd`

2. Haz clic en **Settings** (Configuración)

3. En el menú lateral, busca **Pages** (Páginas)

4. En **Source** (Fuente), selecciona:
   - **Branch**: `main` (o `master`)
   - **Folder**: `/docs`

5. Haz clic en **Save** (Guardar)

6. GitHub generará automáticamente la URL de tu sitio. Normalmente será:
   ```
   https://yohan201609-oss.github.io/playa-rd/
   ```

7. Espera unos minutos (2-5 minutos) para que GitHub procese los cambios.

## Acceder a los documentos

Una vez configurado, podrás acceder a:

- Página principal: `https://yohan201609-oss.github.io/playa-rd/`
- Política de Privacidad: `https://yohan201609-oss.github.io/playa-rd/politica-privacidad.html`
- Términos de Servicio: `https://yohan201609-oss.github.io/playa-rd/terminos-servicio.html`

## Actualizar los documentos

Para actualizar los documentos:

1. Edita los archivos HTML en esta carpeta
2. Haz commit y push a GitHub
3. Los cambios se reflejarán automáticamente en GitHub Pages (puede tardar unos minutos)

## Nota

Los archivos `.md` (markdown) originales se mantienen en esta carpeta para referencia, pero GitHub Pages mostrará los archivos HTML.
